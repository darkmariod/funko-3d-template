Deploy en Vercel

  1. 1. Generá el config.js con el botón de arriba
  2. 2. Reemplazá el archivo config.js en la raíz del proyecto
  3. 3. Subí cualquier archivo .glb o de audio a sus carpetas en /public/
  4. 4. Ejecutá vercel o conectá el repo desde el dashboard de Vercel
  5. 5. Copiá la URL del despliegue y pegala arriba
  6. 6. Generá el QR y descargalo para el cliente
